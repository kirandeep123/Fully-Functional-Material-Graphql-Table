SearchMetrics Gmbh Search and Manage Keywords

## The idea behind the App

1. Add Category
2. Remove Category
3. Add Keyword
4. Remove Keyword

Install it and run:
```sh
cd SearchMetrics Coding Task
npm install 
npm start
```
Install and run json graphql server 
```sh
cd graphql-dummy-server
npm install
node server.js
```
